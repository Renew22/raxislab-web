Como usar esta guia

El secreto de los buenos prompts

Un prompt mal escrito da una respuesta generica. Un prompt bien escrito da una respuesta lista para publicar. La diferencia esta en el contexto que le das a la IA.

Formula maestra: ROL + CONTEXTO + TAREA + FORMATO + RESTRICCIONES

Ejemplo: "Actua como copywriter experto en restaurantes [ROL]. Tengo un restaurante italiano en Madrid con ticket de 35EUR [CONTEXTO]. Escribe un caption para Instagram [TAREA] de maximo 150 caracteres [FORMATO] sin usar palabras como 'autentico' o 'unico' [RESTRICCIONES]."
